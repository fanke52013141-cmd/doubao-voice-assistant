# Voice Sync - 手机语音输入直达电脑光标

## 📋 项目概述

这是一个局域网内的文字同步工具，允许用户在手机上输入文字（包括语音输入），然后**自动输入到电脑上光标所在的任意输入框**中。

### 核心功能
- 手机端网页输入文字 → 自动出现在电脑端光标位置
- 支持任何能输入文字的应用（记事本、Word、微信、浏览器、IDE等）
- 电脑端窗口置顶，可开关自动输入模式

## 🏗️ 技术架构

```
┌─────────────┐     WebSocket      ┌──────────────┐     模拟粘贴      ┌──────────────┐
│  手机浏览器  │ ────────────────→ │  Flask 服务器 │ ────────────────→ │  电脑客户端   │
│  (sender)   │                    │  (server.py) │                    │  (client.py) │
└─────────────┘                    └──────────────┘                    └──────────────┘
      ↓                                   ↓                                   ↓
  输入文字                           广播消息                          复制到剪贴板
  点击发送                                                            模拟 Ctrl+V
                                                                      文字出现在光标处
```

## 🛠️ 技术栈

| 组件 | 技术 | 说明 |
|------|------|------|
| 服务器 | Flask + Flask-SocketIO | 处理 HTTP 和 WebSocket 通信 |
| 异步支持 | eventlet | 支持 WebSocket 异步处理 |
| 手机端 | HTML + Socket.IO Client | 纯网页，无需安装 App |
| 电脑客户端 | PyQt5 | 桌面 GUI 窗口 |
| 自动输入 | pyautogui + pyperclip | 模拟键盘输入 |

## 🔧 关键技术实现

### 1. 服务器端 (server.py)

```python
from flask import Flask, render_template
from flask_socketio import SocketIO, emit

app = Flask(__name__)
socketio = SocketIO(app, cors_allowed_origins="*", async_mode='eventlet')

@app.route('/')
def sender():
    return render_template('sender.html')

@socketio.on('send_text')
def handle_send_text(data):
    text = data.get('text', '')
    if text:
        emit('receive_text', {'text': text}, broadcast=True)

if __name__ == '__main__':
    socketio.run(app, host='0.0.0.0', port=56789)
```

### 2. 自动输入核心逻辑 (client.py)

```python
import pyperclip
import pyautogui

def on_text_received(self, text):
    if self.auto_input_mode:
        # 复制到剪贴板
        pyperclip.copy(text)
        # 模拟 Ctrl+V 粘贴
        pyautogui.hotkey('ctrl', 'v')
```

### 3. 手机端发送 (sender.html)

```javascript
const socket = io();

function sendText() {
    const text = textInput.value.trim();
    if (text && socket.connected) {
        socket.emit('send_text', { text: text });
        textInput.value = '';
    }
}
```

## 🚀 安装步骤

### 1. 环境准备
```bash
# 安装 Python 3.10+
# 下载地址: https://www.python.org/downloads/
# 安装时勾选 "Add Python to PATH"
```

### 2. 安装依赖
```bash
cd voice-sync
pip install -r requirements.txt
```

### 3. 启动服务
```bash
# 终端1: 启动服务器
python server.py

# 终端2: 启动客户端
python client.py
```

### 4. 手机连接
- 确保手机和电脑在同一 WiFi
- 查看客户端显示的 IP 地址
- 手机浏览器访问 `http://电脑IP:56789`

## 💡 使用方法

1. 启动 `server.py` 和 `client.py`
2. 确保客户端窗口的 ⌨️ 按钮是**绿色**（自动输入开启）
3. 在电脑上点击你要输入文字的地方（光标就位）
4. 在手机上输入文字，点击发送
5. 文字自动出现在光标位置！

---

## ⚠️ 注意事项

1. **防火墙**: 首次使用可能需要允许 Python 通过防火墙
2. **同一网络**: 手机和电脑必须在同一局域网
3. **焦点位置**: 发送前确保电脑光标在目标输入框内
4. **剪贴板**: 自动输入会使用剪贴板，会覆盖原有内容
5. **VPN/代理**: 如果有 VPN 软件，可能需要关闭或配置排除局域网

## 🔍 常见问题

| 问题 | 解决方案 |
|------|---------|
| 手机访问不了 | 检查防火墙、确认同一WiFi、关闭VPN |
| PyQt6 报错 | 改用 PyQt5（已在代码中修改） |
| 自动输入无效 | 检查光标是否在输入框、⌨️ 按钮是否绿色 |
| 中文输入问题 | 使用剪贴板+粘贴方式，已解决 |
