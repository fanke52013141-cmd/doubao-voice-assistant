# 语音输入助手 - 使用指南

## 换电脑使用步骤

### 一、环境准备

1. **安装 Python**
   - 前往 python.org 下载安装 Python 3.10 或更高版本
   - 安装时勾选「Add Python to PATH」选项

2. **复制项目文件夹**
   - 将整个 voice-sync 文件夹复制到新电脑

3. **安装依赖库**
   - 打开命令提示符（按 Win+R，输入 cmd）
   - 进入项目文件夹，运行：pip install -r requirements.txt（是这个文件的依赖库，不是这个文件）

---

### 二、日常使用

1. **启动服务器**
   - 双击项目文件夹中的 server.py
   - 或在命令提示符中运行：python server.py
   - 保持服务器窗口打开

2. **启动客户端**
   - 双击项目文件夹中的 client.py
   - 或双击桌面快捷方式（如已创建）

3. **获取手机端地址**
   - 客户端界面顶部会显示「手机访问: xxx.xxx.xxx.xxx:56789」
   - 点击该地址可直接复制

4. **手机端连接**
   - 确保手机和电脑连接同一个 WiFi
   - 手机浏览器打开复制的地址
   - 开始语音输入并点击发送

5. **电脑端复制**
   - 内容会实时显示在客户端窗口
   - 点击「复制」按钮，内容自动复制到剪贴板并清空

---

### 三、创建桌面快捷方式

双击运行项目文件夹中的 create_shortcut.py 即可在桌面创建快捷方式。

---

### 四、注意事项

- 手机和电脑需连接同一个 WiFi
- 每次使用前先启动服务器
- 更换网络后手机端地址会变化，需重新查看客户端显示的地址

---

### 五、防火墙设置（首次使用）

如果手机无法访问，需要关闭防火墙：

1. 按 Win+I 打开「设置」
2. 点击「隐私和安全性」→「Windows 安全中心」→「防火墙和网络保护」
3. 查看当前网络类型（显示「使用中」的那个，通常是「公用网络」或「专用网络」）
4. 点击进入该网络类型
5. 将「Microsoft Defender 防火墙」开关关闭
6. 使用完毕后建议重新打开防火墙
